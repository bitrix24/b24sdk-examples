# คำแนะนำในการใช้และกำหนดค่าการดำเนินการ "การจัดการนวัตกรรม"

## คำอธิบายการดำเนินการ

## **การกำหนดค่าการดำเนินการ "การจัดการนวัตกรรม"**

### 1. การเพิ่มการดำเนินการไปยังกระบวนการธุรกิจ
- ไปที่ส่วน **"กระบวนการธุรกิจ"** ใน Bitrix24 ของคุณ
- สร้างกระบวนการใหม่หรือเปิดกระบวนการที่มีอยู่
- ค้นหาการดำเนินการ **"App Marketplace"** ในรายการการดำเนินการที่มีอยู่และเพิ่มลงในกระบวนการ

### 2. การค้นหาและเลือกแอป
- เปิด **App Marketplace** ผ่านอินเทอร์เฟซของ Bitrix24
- ใช้ตัวกรองและหมวดหมู่เพื่อค้นหาแอปที่ตรงกับความต้องการทางธุรกิจของคุณ (เช่น การตลาด การวิเคราะห์ การจัดการโครงการ)
- ตรวจสอบคำอธิบายแอป คะแนน และความคิดเห็นของผู้ใช้

### 3. การติดตั้งและกำหนดค่าแอป
- คลิก **"ติดตั้ง"** สำหรับแอปที่เลือก
- ทำตามคำแนะนำบนหน้าจอเพื่อทำการติดตั้งให้เสร็จสิ้น
- กำหนดค่าการบูรณาการแอปกับข้อมูลและกระบวนการของคุณใน Bitrix24 (เช่น การเชื่อมต่อ CRM การตั้งค่า API)

### 4. การบูรณาการกับกระบวนการธุรกิจ
- ใช้แอปที่ติดตั้งเพื่อทำให้งานต่าง ๆ เป็นอัตโนมัติ (เช่น การซิงโครไนซ์ข้อมูลอัตโนมัติ การส่งการแจ้งเตือน)
- ตั้งค่าทริกเกอร์และการดำเนินการในกระบวนการธุรกิจเพื่อใช้ประโยชน์จากฟังก์ชันของแอป

### 5. การจัดการและการอัปเดต
- ตรวจสอบการอัปเดตของแอปที่ติดตั้งเป็นประจำเพื่อให้แน่ใจว่ามีความเข้ากันได้และความปลอดภัย
- ลบแอปที่ไม่ได้ใช้งานเพื่อเพิ่มประสิทธิภาพการทำงานของระบบ
