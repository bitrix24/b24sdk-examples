# 使用和配置「數據分析」操作的指南

## 操作描述

## **配置「數據分析」操作**

### 1: 進入工作流程設計器
- 登錄您的 Bitrix24 帳戶。
- 導航至 **CRM** > **自動化** > **業務流程**。
- 點擊 **創建業務流程** 或選擇現有的流程進行編輯。

### 2: 添加銷售管理操作
- 在工作流程設計器中，從可用操作列表中拖放 **銷售管理** 操作。
- 將操作放置在您希望其在工作流程中發生的位置。

### 3: 配置操作
- 潛在客戶追蹤：
  - 定義您的銷售管道階段。
  - 設置潛在客戶狀態變更的通知。
- 績效分析：
  - 選擇您想要追蹤的指標（例如，轉化率、交易價值）。
  - 設置定期績效審查的自動報告。
- 任務自動化：
  - 識別重複性任務（例如，發送後續電子郵件）。
  - 創建自動化規則來處理這些任務。
- 交易管理：
  - 設置交易階段和進展標準。
  - 為每個交易分配任務和截止日期給團隊成員。

### 4: 保存並激活
- 檢查您的設置以確保一切配置正確。
- 點擊 **保存**，然後點擊 **激活**，使銷售管理操作在您的工作流程中生效。
