# 使用和配置“客户支持”操作的说明

## 操作描述

## **配置“客户支持”操作**

### 1. 将操作添加到业务流程中
- 打开您想要添加“客户支持”操作的业务流程。
- 在操作面板中，找到“客户支持”部分，并将操作拖动到业务流程画布上。

### 2. 设置集成
- 安装应用后，访问其设置。
- 根据应用的说明配置与 Bitrix24 的集成。

### 3. 字段映射
- 在配置“客户支持”操作时，将 Bitrix24 字段映射到所选应用中的字段。
- 这允许数据在 Bitrix24 和第三方应用之间传输。

### 4. 设置条件
- 确定“客户支持”操作应在哪些条件下执行。
- 这使操作能够根据特定事件或标准自动执行。
